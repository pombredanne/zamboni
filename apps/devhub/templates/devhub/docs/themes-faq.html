{{ _('Introducing Background Themes') }}

  1. {{ _('What are Background Themes?') }}

    {% trans browse_url=url('browse.personas') %} Background themes are easy-to-install, easy-to-use skins that change the look of your Firefox web browser. There are hundreds of thousands community-designed themes to choose from, with more added every day. Start looking here for designs that fit your style. {% endtrans %}

  2. {{ _('What kind of computers and operating systems do Background Themes work with?') }}

    {% trans %} Background Themes work with any type of computer that has Firefox installed. This includes Mac, Linux, Windows, and Android platforms. {% endtrans %}

  3. {{ _("What's the difference between Background Themes and Complete Themes?") }}

    {% trans %} Complete Themes transform the look of your browser with styles for the window frame, address bar, buttons, tabs, and menus. {% endtrans %}

    {% trans %} Background Themes change just the header and footer areas of your browser. You can instantly preview a design and install it in one click without needing to restart. {% endtrans %}

{{ _('Installing Background Themes') }}

  1. {{ _('How do I add Background Themes to my Firefox?') }}

    {% trans %} In just a few seconds, you can install a theme and transform the look of your Firefox web browser. Preview any theme by simply mousing over it; the design appears in your browser header so you can "try it on." When you're ready, just click the green Add to Firefox button. Once a theme is installed, it is saved in the Add-ons Manager, where you can enable or disable them as you wish. Simply go to the Tools > Add-ons menu and click on the Appearance tab. {% endtrans %}

  2. {{ _('Are Background Themes free?') }}

    {% trans %} Yes. Themes are free to download and free to use. {% endtrans %}

{{ _('Using Background Themes') }}

  1. {{ _('How do I change my current Background Theme?') }}

    {% trans %} There are two easy ways to change your theme. You can either: {% endtrans %}

    • {% trans gallery_url=url('browse.personas') %} Visit the gallery, choose your favorite theme, and click on your choice to instantly change the look of your browser. {% endtrans %}
    • {% trans %} Once a theme is installed, it is saved in the Add-ons Manager, where you can enable or disable them as you wish. Simply go to the Tools > Add-ons menu and click on the Appearance tab. {% endtrans %}

  2. {{ _('If I have a Complete Theme installed, will Background Themes still work?') }}

    {% trans %} Yes. If you have a non-default Complete Theme installed, Firefox will automatically switch to a default theme in order to allow you to use your choice of Background Theme designs. {% endtrans %}

  3. {{ _('How do I undo a Background Theme?') }}

    {% trans %} To undo the design you most recently selected, go to Tools > Add-ons in the menu bar and select the Appearance tab. Next to the theme you want to undo, click the Disable button. {% endtrans %}

  4. {{ _('How do I provide feedback?') }}

    {% trans forum_url='https://forums.mozilla.org/addons/viewforum.php?f=30' %} We're always looking for ways to improve themes, and would love to hear what you think. Visit the themes forum to send us your feedback. {% endtrans %}

{{ _('Creating Your Own Themes') }}

  1. {{ _('Can I create my own Theme?') }}

    {% trans tutorial_url=url('devhub.docs', 'themes') %} Absolutely! All you need to do is create two graphics files in your favorite graphics editing program (e.g., Photoshop). To get started read more about how to create a theme. {% endtrans %}

  2. {{ _('Are there any content guidelines?') }}

    {% trans tos_url='https://marketplace.firefox.com/developers/docs/policies/agreement' %} Yes. We are big fans of creativity, but want to ensure that the art displayed in the Themes gallery meets a basic set of guidelines. Please review the Terms of Service for authoritative and complete language, but the following gives you an overview of the requirements: {% endtrans %}

    • {% trans %} You either own all the elements of your design OR you are authorized worldwide to reproduce and distribute them (and allow others to do so) by the owner or the law. {% endtrans %}
    • {% trans %} Your design does not contain sexually explicit or offensive content. {% endtrans %}
    • {% trans %} Your design does not include identifiable elements of any non-celebrity person(s) other than yourself or individuals who have given you permission to do so. {% endtrans %}
    • {% trans %} Your design does not depict violence or Nazi imagery, nor is it discriminatory or hateful. {% endtrans %}
    • {% trans %} Your design does not relate to gambling. {% endtrans %}
    • {% trans %} Your design does not violate any applicable law or regulation or the rights of any person or entity. {% endtrans %}

      {% trans %} Also, note that your design needs to be suitable for the Background Themes format (see guidelines), of usable quality, and not duplicative of a prior submission. Finally, while Mozilla does provide guidelines for submissions, Mozilla retains the discretion to reject or remove designs for any reason. In some cases, this is necessary for legal reasons, but it also allows us to maintain the integrity of the site. {% endtrans %}

      {% trans tutorial_url=url('devhub.docs', 'themes', 'tutorial') %} Learn more about how to create cool theme designs. {% endtrans %}

  3. {{ _('Do I still retain ownership over artwork that I upload?') }}

    {% trans %} Yes. Your artwork will always be your artwork. {% endtrans %}

  4. {{ _('After creating a Background Theme, can I edit the design?') }}

    {% trans dashboard_url=url('devhub.addons') %} Yes. If at any point you need to edit or delete your theme design after it has been submitted to the gallery, you can access it from the Submissions dashboard. From there, you can view your current designs. By clicking on each, you can make changes to each design's settings. {% endtrans %}

  5. {{ _("What's the maximum file size allowable for my Background Theme?") }}

    {% trans %} The theme you upload may not exceed 300 KB for the header or the footer image. {% endtrans %}
